Mind the Sensors

Sensor awareness, calibration, and robust multimodal monitoring

Abstract

Mind the Sensors investigates the reliability, calibration, and integration of heterogeneous sensor systems for robust environmental monitoring. The project develops methods to detect sensor drift, fuse multimodal data, and provide tools for easy deployment and diagnostics.
